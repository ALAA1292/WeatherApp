
# 🌤️ Weather App

A simple and responsive **Weather Forecast Web Application** that allows users to search for any city and view the current weather along with a 3-day forecast.  
The app can also detect the user's **current location** and display weather data automatically.

## 🚀 Live Demo
🔗weather-app-6qfc.vercel.app

---

## ✨ Features

- 🌍 Search weather by **city name**
- 📍 Detect user's **current location using Geolocation API**
- 📅 Display **3-day weather forecast**
- 🌡️ Show **temperature, weather conditions, and icons**
- 📱 **Responsive design** using Bootstrap
- ⚡ Fast and lightweight frontend

---

## 🛠️ Built With

- **HTML5**
- **CSS3**
- **Bootstrap**
- **JavaScript (Vanilla JS)**
- **WeatherAPI**

---

## ⚙️ How It Works

1. The user enters a **city name** in the search field.
2. The application sends a request to the **WeatherAPI**.
3. Weather data is fetched and displayed dynamically.
4. If location access is allowed, the app automatically loads weather data for the user's **current location**.
